My GA4 Notes (what I did)

I made this site on GitHub Pages so I could practice Google Analytics (GA4) on something I control. Then I created a GA4 property + web data stream and added the GA4 tracking code to this page.

  • GA4 is installed (Measurement ID: G-DFMWZB1QEF)
  • Button tracking: when the CTA button is clicked, it sends an event called cta_click
  • Link tracking: when the LinkedIn link is clicked, it sends an event called outbound_click
  • Conversion (Key event): I marked cta_click as a Key event in GA4
  • Funnel: I built a funnel in GA4 Explore: page_view → cta_click → outbound_click
